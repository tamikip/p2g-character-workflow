# Prompt Template: CG Generation

## Intent
Generate CG-style outputs that keep the original character identity stable.

## Prompt Template
Using the reference character as identity anchor, generate a CG-style image.
Preserve face structure, hairstyle, silhouette, and signature outfit details.
Keep the character recognizable as the same person.
Allow richer rendering, lighting, and composition, but avoid identity drift.
No unrelated character redesign, species change, or costume replacement.
