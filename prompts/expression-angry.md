# Expression Prompt: Angry

## Intent
Generate an "angry" expression while preserving the same character identity from the reference image.

## Prompt Template
Keep the same character identity, facial structure, hairstyle, clothing cues, and overall style from the reference.
Change only the expression to anger.
Eyebrows lowered, tighter mouth shape, stronger facial tension.
Do not change age, gender presentation, body type, or core design details.
Maintain consistency and avoid exaggerated distortion.
