# Expression Prompt: Surprise

## Intent
Generate a "surprise" expression while preserving the same character identity from the reference image.

## Prompt Template
Keep the same character identity, facial structure, hairstyle, clothing cues, and overall style from the reference.
Change only the expression to surprise.
Eyes wider, eyebrows lifted, mouth slightly open in a natural surprised reaction.
Do not change age, gender presentation, body type, or core design details.
Maintain consistency and avoid adding unrelated props or background distractions.
