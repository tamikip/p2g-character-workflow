# Expression Prompt: Surprise

## Goal
Generate a transparent-background surprise expression while keeping the same character identity.

## Current Template
请保持同一角色的人设、发型、五官、服装、配饰、配色与画风一致。  
必须是单人角色，不要新增人物。  
不要改变角色年龄感、体型、服装款式、发型结构和标志性配饰。  
基于抠图，生成惊讶表情，透明背景。表情重点：眼睛睁大、眉毛上扬、嘴巴自然张开。  
只允许修改表情，不要修改姿态、发型、服装、配饰和角色设定。  
输出透明背景 PNG，主体完整，边缘干净。  
负面：多人、文字、水印、裁切、畸形手、多肢、模糊、低清晰度、换装、变发型、变配饰、人物不一致。
